# RouteForge Full Game Engine

Plain static site: HTML + CSS + JavaScript.

Vercel settings:
- Framework Preset: Other
- Build Command: leave blank
- Output Directory: .
- Install Command: leave blank/default

This version includes:
- Real field objects
- Basket/gate collision
- Collectable pieces
- Inventory capacity
- Gate unlock
- Basket scoring
- Basic ranking points
- Shooter trajectory simulation
- Motor acceleration/spin-up
- Field-centric and robot-centric TeleOp


Update:
- TeleOp movement is now continuous. Hold WASD/QE instead of spamming buttons.
- Driver buttons support hold-to-move with pointer/touch.
- The yellow/orange grid has been removed from the field surface.
- Field layout still needs exact dimensions/positions from the real target map before it can be called accurate.
