Command Help

This simulator uses simple FTC-style commands. It is not the real SDK, but it teaches the logic.

  • SET_START x y heading sets robot pose on a 144in × 144in field.
  • FORWARD inches power drives in the current heading.
  • BACK inches power drives backward.
  • STRAFE inches power moves sideways.
  • TURN degrees power rotates the robot.
  • INTAKE seconds power uses intake RPM.
  • SHOOT shots power uses shooter RPM.
  • LIFT inches power uses lift RPM.
  • MARK label drops a waypoint.
  • TELEOP_DRIVE f s t seconds power simulates driver movement.
  • BIND A INTAKE 1.0 0.8 maps a gamepad button.

Keyboard: W/A/S/D = drive, Q/E = turn, Space = run auto, R = reset, 1/2/3/4 = A/B/X/Y. Turn on Driver Mode if you want WASD to control the robot even after clicking around the page.
