Command Help

This is a custom FTC-style robot simulator. It is not the real SDK, but it models the ideas: drivetrain, intake, shooter, lift, gate, scoring, and ranking points.

  • SET_START x y heading sets robot pose on a 144in × 144in field.
  • FORWARD inches power drives in the current heading.
  • BACK inches power drives backward.
  • STRAFE inches power moves sideways.
  • TURN degrees power rotates the robot.
  • INTAKE seconds power collects nearby pieces if intake is in range.
  • OPEN_GATE unlocks stored game pieces.
  • SHOOT target power angle simulates shooter trajectory. Target can be RED or BLUE.
  • SCORE_BASKET scores carried pieces near a basket.
  • LIFT inches power simulates lift/arm movement.
  • MARK label drops a waypoint.
  • TELEOP_DRIVE f s turn seconds power simulates driver movement.
  • BIND A INTAKE 1.0 0.8 maps a gamepad button.

Keyboard: W/A/S/D = drive, Q/E = turn, Space = run auto, R = reset, 1/2/3/4 = A/B/X/Y.
